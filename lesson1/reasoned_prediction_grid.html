Writing a Reasoned Prediction

Testable question:
Change this
Measure this
Relationship: Description here
Cause increase / decrease
more / less
with / without
condition 1 versus condition 2
Effect (result) increase / decrease / stay the same
more / less / no difference
Prediction:
If I do this
then this will happen
Reason:
This is because…
Reasoned prediction:
If I do this then this will happen. This is because of this.
