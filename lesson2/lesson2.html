Lesson 2: Planning and Conducting a Fair Test

Glossary Terms

Variable

An aspect of an investigation that can be different or can change.

Independent variable

The one variable deliberately changed during a fair test.

Dependent variable

The result that is observed or measured during a fair test.

Controlled variables

The variables kept the same so the experiment is fair.

Fair test

An experiment where one variable is changed, one is measured, and all others are kept the same.

Planning and Conducting a Fair Test

Learn how scientists identify variables and plan an experiment that is fair.

Learning objective

Success criteria

  1. I can suggest variables that exist in a scientific scenario.
  2. I can identify independent, dependent and controlled variables.
  3. I can describe how I will change, measure and control variables.

Think like a scientist

The fair-test formula

A fair test helps us work out whether changing one thing causes a measurable effect.

1
Change one thing Choose the independent variable.
2
Measure the effect Record the dependent variable.
3
Keep everything else the same Control the remaining variables.

        The three types of variables

        Scientists give different names to the variable they change, the result they measure and the other variables they keep the same.

        Change

        Independent variable

        The one variable that is deliberately changed. This is the possible cause.

        Measure

        Dependent variable

        The effect or result that is observed and measured.

        Keep the same

        Controlled variables

        Everything else that must stay the same so the test remains fair.

        A fair test changes one variable, measures one result and keeps all other variables the same.
